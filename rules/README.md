# StudyBuddy 规则索引

> 按需查阅，不必全部阅读。根据你当前的情况找到对应文件。

---

## 我在做什么？

### 讨论需求 / 提出想法 / 做产品决策
→ [`discussions.md`](./discussions.md)

### 开始开发一个功能 / 修复一个 Bug
→ [`branching.md`](./branching.md)（先看分支规范）

### 提交 PR / 做 Code Review
→ [`pr-review.md`](./pr-review.md)

### 发布新版本
→ [`releases.md`](./releases.md)

### 不知道文档 / 说明 / 记录该放哪里
→ [`repo-structure.md`](./repo-structure.md)

### 作为 AI Agent 参与协作
→ [`agents.md`](./agents.md)

### 远程异步协作约定（时区、响应时间等）
→ [`async-collab.md`](./async-collab.md)

---

## 产品全流程各阶段入口

| 阶段 | 规则文件 |
|------|---------|
| 需求讨论 → 功能定义 | [`discussions.md`](./discussions.md) |
| 技术设计 → 方案评审 | [`discussions.md`](./discussions.md) + [`repo-structure.md`](./repo-structure.md) |
| 开发 → 提测 | [`branching.md`](./branching.md) + [`pr-review.md`](./pr-review.md) |
| 合并 → 发布 | [`releases.md`](./releases.md) |

---

## 规则文件清单

| 文件 | 内容摘要 |
|------|---------|
| [`repo-structure.md`](./repo-structure.md) | 仓库目录约定，内容该放哪里 |

| [`pr-review.md`](./pr-review.md) | PR 模板、Review 流程、合并条件 |
| [`discussions.md`](./discussions.md) | 何时用 Discussion / Issue / Wiki |
| [`releases.md`](./releases.md) | 版本号规范、发布流程、Tag 规则 |
| [`agents.md`](./agents.md) | AI Agent 的协作边界和使用规范 |
| [`async-collab.md`](./async-collab.md) | 异步团队的响应时间和沟通约定 |
